Support

Help for imports, permissions, calendars, and account issues.

TourBuddy helps touring artists and crews import and save routes, compare schedules, calculate sleep windows across time zones, and export reminders to calendar.

Support page for App Store listing and general user help.

What TourBuddy Helps With

  • import and save tour routes
  • review poster or screenshot-based date extraction
  • calculate sleep windows across time zones
  • compare another routing for same-city overlaps
  • export sleep blocks and reminders to calendar

Common Help Topics

Poster import is not working

  • Make sure the image is readable and not a PDF.
  • Phone screenshots are supported, but dense posters may still need review edits.
  • After import, always check the review screen before saving.

Calendar access was denied

  • Open iPhone Settings.
  • Find TourBuddy.
  • Enable Calendar access.
  • Return to the app and try again.

Photo access was denied

  • Open iPhone Settings.
  • Find TourBuddy.
  • Enable Photos access.
  • Return to the app and try the poster import again.

My tour dates imported incorrectly

  • Use the review step to correct dates, cities, or venues before saving.
  • If a poster is especially dense or stylized, try a clearer screenshot or crop.

How do I delete my account?

  • Open the Account screen in TourBuddy.
  • Scroll to the Danger Zone.
  • Tap Delete Account & All Data.
